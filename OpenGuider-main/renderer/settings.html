Panel sohbeti ve Cline için hangi yapay zeka kullanılacağını Sauron otomatik seçer. API anahtarlarını buraya gir — Bridge, VS Code açıkken Cline'a senkronlar.

    Ajan bakiyeleri

    Tahmini maliyet — gerçek fatura ile küçük fark olabilir.

    Maliyet optimizasyonu

    Hangi iş için hangi modelin kullanılacağını otomatik seçer. Panel sohbeti Gemini; günlük kod işleri DeepSeek; zor işler OpenAI. Günlük bütçe aşılırsa model bir kademe düşer.

    Görev özeti bu uzunluğu geçmez; plan adımları korunur.
    Yerel kod agent'a gönderilen dosya parçası limiti.
    Günlük limit aşılırsa Cline için bir kademe daha ucuz model önerilir.

    Ekonomi modu (Ultra)

    Token tasarrufu: tekrarlayan görevlerde küçük özet, gereksiz soruları atlama, yerel Ollama önerisi.
