# Scaffold Instructions

This project was generated from the Sauron **corporate-nextjs** template.

## Quick start

```bash
npm install
npm run dev
```

Then open **http://localhost:3000** in your browser.

## Customize

Edit `.sauron/web-brief.json` in the workspace root for company metadata, or update components directly.

Brand placeholders injected at scaffold time:

- `{{COMPANY_NAME}}`
- `{{TAGLINE}}`
- `{{PRIMARY_COLOR}}`
- `{{ACCENT_COLOR}}`

## Production

```bash
npm run build
npm run start
```
