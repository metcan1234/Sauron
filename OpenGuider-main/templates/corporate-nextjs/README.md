# {{COMPANY_NAME}} — Corporate Website

Next.js 14 corporate template scaffolded by Sauron Web Studio.

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run start` | Start production server |

## Pages

- `/` — Home (Hero, services, stats, testimonial, CTA)
- `/about` — Company overview
- `/services` — Service offerings
- `/contact` — Contact form

## Deployment

Deploy to [Vercel](https://vercel.com) or any Node.js host. A minimal `vercel.json` is included.
